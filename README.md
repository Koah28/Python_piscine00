## 📂 Estructura del repositorio

```bash
ex0/
ex1/
ex2/
ex3/
ex4/
ex5/
ex6/
ex7/
```

Cada carpeta contiene un ejercicio con su correspondiente función en Python.

---

## 🧠 Ejercicios incluidos

### 🌱 ex0 — Hello Garden

Primer contacto con Python → imprimir texto.

### 🪴 ex1 — Garden Name

Uso de `input()` y `print()`.

### 📐 ex2 — Garden Plot Area

Cálculos básicos y uso de `int()`.

### 🍅 ex3 — Harvest Total

Suma de múltiples valores.

### ⏳ ex4 — Plant Age Check

Condicionales (`if / else`).

### 💧 ex5 — Water Reminder

Lógica aplicada a un caso práctico.

### 🔁 ex6 — Count to Harvest

Bucles y recursividad.

### 📦 ex7 — Seed Inventory

Type hints + manejo de strings.

## ⚙️ Cómo funciona

* Cada archivo contiene **solo una función**
* No hay `main`
* No se ejecuta directamente
* Se prueba con un archivo externo (`main.py`)
